# Multi-Chain Transactions + Crypto

Pangea ID can be used to sign transactions and manage crypto assets on multiple blockchains.

The following protocols and blockchains are currently supported:

1. [WalletConnect](wallet-connect.md) on
   1. Ethereum
   2. Polygon
   3. Ethereum Sepolia (Testnet)
2. [Antelepe Signing Requests](antelope-sigining-request-esr.md) (Pangea ID can be used as a replacement for Anchor wallet)
   1. Pangea Mainnet
   2. Pangea Testnet (Testnet)

To see testnet coins in a user wallet, go to the app settings and turn on developer mode.
